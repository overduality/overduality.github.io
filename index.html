Muvo

Muvo is an edge-AI speech assessment engine designed to bridge the gap between textbook learning and real-world communication confidence in high-stakes travel scenarios. Built with a Multi-Modal Decision Fusion architecture, the system overcomes the accuracy plateaus of traditional sound classification by intelligently pairing Core ML prosodic analysis with Apple's Speech Framework for lexical validation, successfully elevating system accuracy from 62% to 85%. By engineering a robust pipeline that processes over 9,300 augmented samples with sub-200ms latency via Accelerate (vDSP), I transformed a technical challenge into a high-performance, empathetic product that provides instant, encouraging feedback while preserving user privacy through entirely on-device processing.
